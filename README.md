# Insomnia Cure - Sleep Tracking Application

A Flask-based web application to help users track sleep patterns and improve sleep quality.

## Features
- Sleep tracking and logging
- Sleep pattern visualization
- AI chatbot for sleep advice
- Relaxation tools
- Data export

## Setup
1. Create virtual environment: `python -m venv venv`
2. Activate: `.\venv\Scripts\Activate.ps1`
3. Install dependencies: `pip install -r requirements.txt`
4. Set up MySQL database using `database.sql`
5. Run: `python app.py`

## Technologies
- Flask
- MySQL
- Bootstrap 5
- Chart.js
